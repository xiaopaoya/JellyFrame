# Samples

这里保存页面样例和 app package 样例。原生桌面工具源码位于 `../tools/native`。

- `apps/packages/`：带 `jellyframe.app.json` 的完整 JellyFrame source package。
- `apps/loose/`：用于聚焦检查 scripting/rendering 的小型散文件 app fixture。
- `pages/modern/`：用于观察合理降级的现代 HTML/CSS 独立页面。
- `scripts/classic/`：可选 JerryScript 壳使用的 classic-script 探针。
- `fonts/bitmap/`：字体工具 smoke test 使用的小型 bitmap-font 输入。

样例应保持有代表性但受控。app 作者起始模板位于 `../tools/templates/apps`；
samples 用于验证、截图和回归。
