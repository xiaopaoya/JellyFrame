# Samples

Sample pages and app packages live here. Native desktop tools live in
`../tools/native`.

- `apps/packages/`: complete JellyFrame source packages with
  `jellyframe.app.json`.
- `apps/loose/`: small loose-file app fixtures used for focused scripting and
  rendering checks.
- `pages/modern/`: standalone modern HTML/CSS pages that exercise graceful
  degradation.
- `scripts/classic/`: classic-script probes for the optional JerryScript shell.
- `fonts/bitmap/`: small bitmap-font inputs for font tooling smoke tests.

Keep samples representative but bounded. App-author starting points belong in
`../tools/templates/apps`; samples are for validation, screenshots and regressions.
